プログラム復習コンテンツ

dist()関数

サンプルコードの結果

サンプルコード

　　 void setup(){

　　 　size(480, 120);

　　 　stroke(0, 102);

　　 }

　　 void draw(){

　　 　float weight=dist(mouseX, mouseY, pmouseX, pmouseY);

　　 　strokeWeight(weight);

　　 　line(mouseX, mouseY, pmouseX, pmouseY);

　　 }

説明文

　dist()関数は2点間の距離を求める関数。

　pmouseXやpmouseYと組み合わせることで読み込むごとにdist()関数の値が変動する。

　この変動する値をstrokeWeight()に渡すことで線の太さが変わる線を描くことができる。

　マウスを早く動かすことで2点の距離が大きくなり線は太くなりゆっくり動かすことで2点の距離は短くなり値が小さくなるため太さも細くなる。
